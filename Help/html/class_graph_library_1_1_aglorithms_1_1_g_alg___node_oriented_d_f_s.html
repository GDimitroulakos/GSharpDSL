Detailed Description

This algorithm ( visitor ) performs a DFS traversal over a graph (directed or undirected ). It records the paths from which the algorithm passes by coloring nodes along them. The algorith works also for non-connected graphs 1) White (0) : Not yet visited 2) Gray (1) : Visited but not all of its neighbours 3) Black (2) : Visited and all of its neighbours
