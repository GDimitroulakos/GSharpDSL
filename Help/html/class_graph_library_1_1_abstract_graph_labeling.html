Detailed Description

This class holds the labels for a specific type of elements (edge , nodes, graphs ) of a graph. I use two dictionaries for both mapping to gain in query speed. This class gives the opportunity that diffrent graph clients can assign diffrent labels to the graph nodes, edges etc. The graph client ( contructor ) creates a subclass of AbstractGraphLabeling<T> and assigns labels nodes. The instance of this class exists inside the CGraph class and the client can add a new labeling of graph elements through the AddGraphNodeLabelling() method.
